# Бот расписания СП ЦПСУ "ЭНЕРГИЯ"

Телеграм-бот, который автоматически:

- получает Word-файлы с расписанием из Gmail;
- конвертирует их в PDF;
- переименовывает по дате выполнения (сегодняшнее письмо = расписание на завтра);
- отправляет уведомления в Telegram;
- хранит архив расписаний и позволяет их просматривать.

## Стек

- Python 3.10+
- `python-telegram-bot` (polling)
- `imap-tools` / `imaplib` (здесь используется стандартный `imaplib`)
- `docx2pdf` для конвертации `.doc`/`.docx` → PDF (требуется установленный Microsoft Word в Windows)
- `sqlite3` для хранения метаданных

## Установка

1. Клонируйте/скопируйте проект:

```bash
cd bot_rasp
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

2. Создайте Telegram-бота через `@BotFather` и получите токен.

3. Настройте доступ к Gmail:

- включите 2FA;
- создайте пароль приложения (App Password) для IMAP;
- убедитесь, что IMAP включен в настройках Gmail.

4. Заполните `config.py`:

- `TELEGRAM_BOT_TOKEN`
- `IMAP_EMAIL`
- `IMAP_PASSWORD`
- при необходимости другие пути/настройки.

## Запуск

```bash
python main.py
```

Бот:

- каждые `MAIL_CHECK_INTERVAL` секунд (по умолчанию 600 = 10 минут) проверяет почту;
- ищет новые непрочитанные письма от `ALLOWED_SENDER` с вложениями `.doc`/`.docx`;
- конвертирует вложения в PDF;
- сохраняет их в папку `archive/ГОД/ММ/Расписание_ГГГГ-ММ-ДД.pdf`;
- рассылает уведомление и PDF всем, кто нажал `/start`.

## Команды бота

- `/start` — приветствие, инструкция и подписка на уведомления.
- `/today` — расписание на сегодня.
- `/tomorrow` — расписание на завтра.
- `/yesterday` — расписание на вчера.
- `/date дд.мм` — расписание на конкретную дату.
- `/list` — список всех доступных расписаний.
- `/help` — краткая справка.

## Замечания

- Дата расписания считается как \(дата_письма + 1\) день.
- Если файл на запрошенную дату еще не загружен, бот сообщает об этом текстом.
- При повторной загрузке расписания на ту же дату файл и запись в базе обновляются.
