# Веб-панель (Vue 3)

Перед первым запуском **соберите фронтенд**:

```bash
cd webapp
npm install
npm run build
cd ..
```

После этого запускайте `python api.py` (или сервис `bot_rasp_web`). Панель будет доступна по основному адресу сервера.

Локальная разработка: `npm run dev` (панель на :5173, запросы к API проксируются на :5000).
